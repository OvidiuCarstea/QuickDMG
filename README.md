QuickDMG
========
An application to create a disk image from a file or a folder with minimal efforts.

## Usage
English :
* http://www.script-factory.net/software/misc/QuickDMG/en/index.html

## Building
Reqirements :
* Mac OS X 10.8 or later.
* Xcode 4.6.2 or later.
* [ModuleLoader]
